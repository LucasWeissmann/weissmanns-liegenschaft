# Weißmanns Liegenschaft 🏖️

Die offizielle Liegenreservierung der Familie Weißmann. Keine Anmeldung nötig.

## Features

- 2 Liegen mit visueller Zeitleiste
- Buchung mit Name und freier Zeitwahl (7:00 - 21:00)
- Echtzeit-Synchronisation (alle sehen Änderungen sofort)
- Überlappungs-Erkennung
- Installierbar als App auf dem iPhone (PWA)

## Tech Stack

- React + Vite + Tailwind CSS
- Firebase Firestore (kostenlos)
- GitHub Pages (kostenlos)

## Setup

Siehe [SETUP.md](./SETUP.md) für die komplette Anleitung.

## Entwicklung

```bash
npm install
npm run dev
```
